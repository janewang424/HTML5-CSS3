Block-level Elements

A block-level element's characteristics:
  • Each one set as one line
  • Can define height, width and margin
  • Default width is 100% of the father block
  • It's a block or box, can contain block-level elements or inline-level elements
There shouldn't be div in p or h1-h6.

Inline-level Elements

An inline-level element's characteristics:
  • Could be a few in a single line
  • Can NOT define height, width and margin
  • Default width is its content width
  • It can only contain text or other inline-level elements.
For Example: This is bold. There shouldn't be a 'a' in one 'a' but can be 'div' in an 'a'

Special Elements

characteristics for special elements like 'img' 'input' 'td':
  • In the same line as the neighbour special elements, but remain space between. Could be more than one special elements.
  • Can define height, width and margin
  • Default width is its content width
